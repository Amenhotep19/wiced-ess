ESS quickstart

Johannes Winkelmann, jwi@sensirion.com

How to install:

1. Create a new app folder called 'ess_demo'
2. Copy ess_demo_c.txt to this folder, rename to ess_demo.c
3. Copy ess_demo_mk.txt to this folder, rename to ess_demo.m
4. Create a new target called <appfolder>.ess_demo-<PLATFORM> download run


Then simply double click on target, and enjoy
